# 🏋️ Fitness Tracker Website

## 📌 Project Description
This project is a simple fitness tracking website developed using HTML, CSS, and JavaScript. It helps users calculate their Body Mass Index (BMI) based on weight and height and provides their fitness status.

---

## 🎯 Objectives
- To develop a basic fitness website
- To calculate BMI using user input
- To understand front-end web development concepts

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 💡 Features
- User-friendly interface
- BMI calculation
- Instant result display
- Lightweight and fast

---

## 🧮 How It Works
1. User enters weight and height
2. System calculates BMI using formula:
   BMI = weight / height²
3. Displays result with status:
   - Underweight
   - Normal
   - Overweight
   - Obese
